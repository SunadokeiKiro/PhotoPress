# PhotoPress アプリのテスト方法

## 準備するもの
1.  **スマートフォン** (Android または iOS)
2.  **Expo Go アプリ** (App Store または Google Play ストアからインストール)
3.  **PC** (このプロジェクトが動作している環境)

## 手順

1.  **サーバーの起動**
    ターミナルで以下のコマンドが実行されていることを確認してください（実行されていない場合は実行してください）:
    ```bash
    npx expo start
    ```

2.  **QRコードの読み取り**
    -   ターミナルに表示されたQRコードを、スマートフォンのカメラ（またはExpo Goアプリのスキャン機能）で読み取ります。
    -   **注意**: PCとスマートフォンは**同じWi-Fiネットワーク**に接続されている必要があります。

3.  **アプリの起動**
    -   スマートフォンのExpo Goアプリ内で、PhotoPressが起動します。
    -   初回起動時はビルドに少し時間がかかる場合があります。

## テスト項目
-   **画像選択**: ギャラリーから画像を選べるか。
-   **Exif削除**: 「オリジナル」サイズを選んだ場合、Exif情報は削除されているか（別のExif確認アプリ等で検証可能）。
-   **リサイズ**: 「1MB」「500KB」などを選んだ際、ファイルサイズがそれ以下になっているか。
-   **共有**: シェアボタンを押して、他のアプリ（LINEやTwitterなど）に画像を送れるか。

## トラブルシューティング
-   **接続できない場合**:
    -   PCとスマホが同じWi-Fiにいるか確認してください。
    -   `npx expo start --tunnel` を試してください（ネットワーク制限がある場合）。
